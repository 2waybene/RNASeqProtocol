DATABASES in mm10

*mm10.fa - 05Nov2019
Chromosome sequences downloaded from UCSC, re-ordered, and assembled to single genome FASTA file with or without
the alt chromosomes.

*mm10.fa.fai - 05Nov2019
Index for reference sequence file, generated by .samtools faidx'.

*mm10.chromSizes -05Nov2019
List of all chromsomes and sizes (tab de-limited). Downloaded from UCSC then re-ordered. 

*mm10.dict
Generated by picard CreateSequenceDictionary.jar v1.69.
java -jar  ~/tools/picard-tools-1.69/CreateSequenceDictionary.jar  REFERENCE=mm10.fa OUTPUT=mm10.dict

*mm10_RefSeq10oct2019.gtf
NCBI RefSeq gene models as downloaded from UCSC Table Browser Nov 05 2019. Genome rowser view of this track notes "Annotation Release NCBI Homo sapiens Annotation Release 109 (2018-03-29)";
I assume the gene models taken from the table browser are also dated March 29 2018.
Transcripts were assigned to Entrez genes based on  release88.accession2geneid &  release96.accession2geneid.gz (source ftp://ftp.ncbi.nlm.nih.gov/refseq/release/release-catalog/).
Entrez gene ID to gene symbols were assigned based on ftp://ftp.ncbi.nlm.nih.gov/gene/DATA/GENE_INFO/Mammalia/Mus_musculus.gene_info.gz.
Transcripts on 'alt' chromosomes and transcripts without an associated Entrez gene were removed from the gtf.

##      annotate with release96
perl  modify_gtf_mm10.pl UCSC_download_mm10_ucsc_NCBIrefSeq.gtf release96.accession2geneid Mus_musculus.gene_info > mm10_ucsc_NCBIrefSeq.gtf

##	modify if -- else condition to output gene_without_annotation
perl  modify_gtf_mm10.pl UCSC_download_mm10_ucsc_NCBIrefSeq.gtf release96.accession2geneid Mus_musculus.gene_info >  gene_without_annotation.txt 

##	annotate with release88
perl  modify_gtf_mm10.pl gene_without_annotation.txt  release88.accession2geneid Mus_musculus.gene_info >> mm10_ucsc_NCBIrefSeq.gtf 

##	Sort the gtf file
sort -k1,1 -k4,4 < mm10_ucsc_NCBIrefSeq.gtf  > temp
mv temp mm10_ucsc_NCBIrefSeq_09Nov2019.gtf
